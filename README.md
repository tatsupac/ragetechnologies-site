# RageTechnologies Corporate Site

Static corporate website for RageTechnologies.

## GitHub Pages

This site is designed to be published from the repository root with GitHub Pages.

- Custom domain: `ragetechnologies.jp`
- Privacy Policy URL: `https://ragetechnologies.jp/privacy.html`

GitHub Pages settings:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`
- Custom domain: `ragetechnologies.jp`
- Enforce HTTPS: enabled
